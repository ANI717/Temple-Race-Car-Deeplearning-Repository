1.0 Install Anaconda (64 Bit, Python 3.7 version or higher)

1.1 Open Anaconda prompt.

1.2 Run the following commands.
	pip install opencv-contrib-python
	conda install pytorch -c pytorch
	conda install torchvision -c pytorch



2.0 Install Keras.
	conda install keras



3.0 Conda Environment
	conda create -n myenv (or, conda create -n myenv python=3.6 scipy=0.15.0 astroid babel)
	conda activate myenv
	conda list
	conda activate (or, conda deactivate)
